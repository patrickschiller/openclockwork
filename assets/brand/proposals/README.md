# OpenClockwork logo proposals

Three chess-clock inspired logo directions for OpenClockwork. Each proposal contains a wide logo with wordmark and a compact favicon candidate.

## Proposal 1: Classic match clock

- Files: `proposal-1-classic-logo.svg`, `proposal-1-classic-favicon.svg`
- Direction: reliable, operational, close to a physical chess clock.
- Best fit: product UI, documentation, and PWA icons where immediate recognition matters most.

## Proposal 2: Split time

- Files: `proposal-2-split-time-logo.svg`, `proposal-2-split-time-favicon.svg`
- Direction: modern dual-timer badge with a stronger open-source/product feel.
- Best fit: app header and small favicons where the shape needs to stay readable.

## Proposal 3: Digital board clock

- Files: `proposal-3-digital-board-logo.svg`, `proposal-3-digital-board-favicon.svg`
- Direction: digital attendance timer with subtle chessboard references.
- Best fit: SaaS-style surfaces and mobile app icons.
